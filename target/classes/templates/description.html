Registration is required to use the application, upon completion of registration please check your email and activate your account. Basically there are 3 registered user in the system already: user, manager and admin. All of them can be logged in with the "pass" password.
Application Features (Full CRUD support):
  • The users are able to:
    • access the list of employees
    • search by name
  • The managers are able to:
    • access the list of employees
    • search by name
    • add new employee to the database
    • update an employee
  • The admins are able to:
    • access the list of employees
    • search by name
    • add new employee to the database
    • update an employee
    • delete an employee

Technologies:

  • Spring Boot
  • Spring Security
  • MySQL
  • Spring Data JPA
  • Hibernate
  • Thymeleaf
